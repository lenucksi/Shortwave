Shortwave Flatpak Repo

This repository provides nightly development builds of Shortwave, an internet radio player with access to over 50,000 stations.

Note: This is a local development fork of Shortwave, originally created by Felix Häcker. It is used to prepare submissions to the upstream project — not intended as a permanent fork. All credit for the original application goes to Felix Häcker and the GNOME community.

Installation

Add the repository and install Shortwave:

flatpak remote-add --user shortwave-nightly https://lenucksi.github.io/Shortwave/index.flatpakrepo
flatpak install --user shortwave-nightly de.haeckerfelix.Shortwave.Devel

Or in one step:

flatpak install --user https://lenucksi.github.io/Shortwave/index.flatpakrepo
